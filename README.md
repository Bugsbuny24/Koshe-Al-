# Koschei — V1 → V20 Product README

## Nedir?

Koschei, kullanıcının ihtiyacını doğal dille alıp bunu doğru iş akışına yönlendiren, planlayan, workspace'e taşıyan, teslim ve revizyon döngüsünü yöneten bir **AI Work Operator** ürünüdür.

Ürünün temel omurgası:

**Chat / Intake → Execution Planning → Workspace / Delivery → Feedback Loop → Intelligence**

Koschei'nin amacı sadece içerik üretmek ya da basit bir chatbot olmak değildir. Amaç; işin ihtiyacını anlamak, bunu yapılandırmak, doğru modüle yönlendirmek ve zamanla daha akıllı hale gelen bir operasyon sistemi kurmaktır.

## Ürün İlkeleri

### 1. Önce çalışan çekirdek
Yeni modül eklemekten önce çalışan omurga korunur.

### 2. Dürüst scaffolding
Scaffolded modüller canlı feature gibi gösterilmez.

### 3. Riskten uzak büyüme
Yüksek regülasyonlu alanlara girilmez.

### 4. Tek ürün, çok kullanım alanı
Ayrı ayrı ürünler değil; tek omurga üstünde genişleyen modüller.

### 5. Yapay zekâ gösterisi değil, iş sistemi
Koschei bir “AI demo” değil, iş akışı ve operasyon sistemi kurar.

## Şu Anki Öncelikli Teknik Gündem

Mevcut repoda çekirdek odak noktaları şunlardır:

1. projects auth / create stabilitesi  
2. execution → project back-link doğrulaması  
3. revision → execution feedback loop doğrulaması  
4. chat conversational onboarding iyileştirmesi  
5. landing polish ve ürün dili tutarlılığı  

Not: V4–V20 için oluşturulmuş SQL skeleton migration dosyaları bu aşamada repo’dan kaldırılmıştır. Gerekirse ileride yeniden üretilecektir.
